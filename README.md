**WhaWeb AI - Dashboard d'Automatisation WhatsApp**
WhaWeb AI est une solution complète de gestion et d'automatisation de la relation client via WhatsApp. Ce projet combine la puissance de Node.js pour la communication en temps réel et l'intelligence artificielle pour des réponses automatiques intelligentes.

 **Fonctionnalités Principales**
Authentification par QR Code : Connexion sécurisée via l'API whatsapp-web.js.

Gestion des Contacts : Système CRUD complet pour gérer vos listes de diffusion avec MongoDB.

Intelligence Artificielle : Intégration avec OpenAI (via n8n ou direct) pour des réponses automatiques basées sur un "System Prompt" personnalisé.

Dashboard Moderne : Interface utilisateur réactive construite avec React et Tailwind CSS.

Campagnes Marketing : Envoi de messages groupés à des segments de contacts spécifiques.

**Stack Technique**
Frontend : React.js, Axios, Lucide React, Tailwind CSS.

Backend : Node.js, Express.js.

Base de données : MongoDB (Mongoose ODM).

API WhatsApp : whatsapp-web.js (Puppeteer sous le capot).
